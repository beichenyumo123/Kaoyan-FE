# 版本更新说明 (v2.3)

> 考研论坛 AI 智能体陪伴舱 — 前端功能更新汇总

---

## 一、AI 答疑页（AiAsk）

### 1.1 导航增强

- **返回首页按钮**：顶栏左侧 Home 图标 → 跳转 `/ai`
- **错题集入口**：顶栏右侧 BookOpen 图标 → 跳转 `/questions`

### 1.2 外部联动

从知识库或错题详情跳入时，自动**创建新会话**（不污染已有对话），上下文通过 `sessionStorage` 传递。

## 二、页面联动

### 2.1 社区推荐 → 知识库 → AI 答疑

```
社区首页 AI 推荐卡片          知识库                     AI 答疑
     │                          │                          │
     ├─ keyword + subject ────→ 自动搜索                  │
     │                          │                          │
     │                     「AI 深入答疑」按钮             │
     │                          └─ question + subject ────→ 新建会话 SSE 回答
```

**涉及文件**：`CommunityHome.vue`、`AiKnowledge.vue`、`AiAsk.vue`

### 2.2 错题详情「不懂模式」

**涉及文件**：`WrongQuestionDetail.vue`、`AiAsk.vue`

- 顶栏「问 AI」按钮 → 进入不懂模式，底部浮现输入条
- **右键选中文本** → 自定义菜单「问问 AI 这段内容」
- 选中原文 **CSS Highlight API 蓝色波浪下划线高亮**
- 自动携带「原题 + 笔记 + 选中上下文 + 用户疑问」跳转 AI 答疑
- 非不懂模式下右键菜单为浏览器原生菜单，不影响复制粘贴

## 三、AI Dashboard — 多智能体陪伴舱

### 3.1 任务 & 干预「查看详情」

| 模块 | 改动 |
|------|------|
| 今日 AI 定制推荐 (Tasks) | 有 `detailMarkdown` 时卡片右侧出现 👁️ 按钮 → 弹通用 AgentDetailModal |
| 智能体主动干预 (Interventions) | 新增「查看详情」按钮 + 行为分析师 fallback（无后端数据时自动跳转 AI 答疑） |

### 3.2 智能体名称对齐 (API v2.3)

`agentName` 从英文改为中文，与左侧「AI 专家学伴团」命名一致：

### 3.3 知识库智能学科识别

## 四、关键技术修复

| 问题 | 根因 | 修复 |
|------|------|------|
| 离开 AiAsk 页面白屏 | 根元素 `transition-all` 与 Vue 路由过渡 `mode="out-in"` 冲突 | 改为 `transition-[gap]`，仅在 gap 属性上过渡 |
| 路由 401 硬刷新白屏 | `setUnauthorizedHandler` 未注册，默认 `window.location.href` | `main.ts` 注册 handler，用 `router.push('/')` 软导航 |
| 右键菜单弹不出来 | `anchorNode.nodeType === 3` (TextNode) 无 `.closest()` 方法 | 判断 nodeType，TextNode 取 `parentElement` |
| Markdown 链接渲染为纯文本 | URL 含空格/中文违反 CommonMark 规范，markdown-it 拒绝解析 | `renderMarkdown()` 预处理：`encodeURI()` 编码链接 URL |
| 编辑按钮不退出不懂模式 | `startEdit` 未重置 `askMode` | `startEdit` 首行添加 `askMode.value = false` |

## 五、API 契约 (v2.3)

### 5.1 `GET /api/ai/tasks`

| 字段 | 类型 | 说明 |
|------|------|------|
| `detailMarkdown` | String \| null | 任务详情（markdown），有则显示「查看详情」按钮 |
| `linkTarget` | String \| null | 详情底部跳转路由 |
| `linkLabel` | String \| null | 跳转按钮文案 |

### 5.2 `GET /api/ai/interventions`

| 字段 | 类型 | 说明 |
|------|------|------|
| `agentName` | String | ⚠️ 改为中文 |
| `detailMarkdown` | String \| null | 干预详情（markdown），行为分析师在此放习题 |
| `linkTarget` | String \| null | 跳转路由 |
| `linkLabel` | String \| null | 跳转按钮文案 |

### 5.3 Markdown 内容规范

- **链接格式**：`[习题名](/ai/ask?question=关键词)` — URL 含空格/中文会被前端 `encodeURI()` 自动编码，无需后端处理
- **习题答案**：用 `<details><summary>📝 点击查看答案</summary>内容</details>` 包裹，实现默认隐藏、点击展开
- **整体渲染**：前端统一走 `renderMarkdown()`（markdown-it + KaTeX + highlight.js），支持 LaTeX、代码高亮、表格、HTML 标签
